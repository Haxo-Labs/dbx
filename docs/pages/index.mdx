# Quick Start

```bash
# Run with Docker
docker run -d --name dbx -p 3000:3000 \
  -e DATABASE_URL=redis://localhost:6379 \
  fnlog0/dbx:latest

# Use the TypeScript SDK
npm install dbx-sdk
```
